# Devi Prasad Portfolio Website

This is a personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- npm (comes with Node.js)

## Getting Started

1.  **Unzip the downloaded file** to a folder on your computer.

2.  **Open a terminal** (Command Prompt, PowerShell, or Terminal) and navigate to the project folder:
    ```bash
    cd path/to/portfolio_source
    ```

3.  **Install dependencies**:
    ```bash
    npm install
    ```

4.  **Start the development server**:
    ```bash
    npm run dev:client
    ```

5.  **Open your browser** and visit:
    http://localhost:5000

## Building for Production

To create an optimized build for deployment:

```bash
npm run build
```

The built files will be in the `dist/public` directory.
